# MAA1999WPF
拿WPF框架写的一个MAA199的UI  
(本来想用Avalonia写的，但没写明白  

现在的状态只是能用，很多东西还不是最优化  
(WPF方面还只是个小白  
欢迎各位大佬贡献代码  

使用的话需要[MAA1999](https://github.com/MaaAssistantArknights/MAA1999)  
然后需要把这个项目编译出的文件放在一个文件夹里
首先先运行一遍命令行版本，生成下config文件  
然后就可以用UI界面了

## 使用的Nuget包
[modern WPF](https://github.com/Kinnara/ModernWpf)  
[.net community toolkit](https://github.com/CommunityToolkit/dotnet)
